#  Basic Calculator (Python)

A simple Python calculator built using conditional statements.
This project takes two numbers and an operator as input and performs basic arithmetic operations.
# Features

Addition

Subtraction

Multiplication

Division

Modulo
# How It Works

The calculator asks for:

First number

Operator (+, -, *, /, %)

Second number

Then it performs the selected operation using conditional statements (if,elif,else)
